
# Pet Simulator Game
This is a Pet Simulator where you can make and take care of a digital pet through keeping track of their hunger, happiness, energy, and various other characteristics. You can feed them, play with them, train them to do certain skills, and do multiple other things.

## How to Use
- Nothing needs to be downloaded. Just run main.py.

## Project Features
- Making Pets
- Taking care of Pets (Hunger, Energy, Happiness, Health, etc.)
- Saving and Loading various Pets
- Pet Shop and Economy (Virtual Currency, Pet Items, Inventory)
- Pet Competitions (Various Events, Leaderboards, Rewards)

## Planned Project Features
- Pet Breeding System (Combining Characteristics, Genetic Traits, Family Trees)
